Quick Start & Support

Share this with table workers and officials.

How to Run a Mat

  • Open the Control Panel and confirm the selected mat matches the mat’s scoreboard.
  • Set wrestler names in the ⚙️ drawer (Match Settings).
  • Choose the period length in the ⚡ drawer (Timer Settings); defaults come from the selected event.
  • Use the Start/Stop/Reset buttons to control the clock.
  • Use the scoring buttons (E1, R2, T3, N2, N3, N4) to update scores.
  • Scoreboard pages will update live for each mat.

Common Setup

  • One laptop or tablet per mat running the Control Panel.
  • One TV/projector per mat running that mat’s Scoreboard.
  • Optional extra device at the head table running the Overview.

Troubleshooting

  • If Status shows “connecting…”, refresh the Control Panel.
  • If a scoreboard doesn’t update, verify the mat number matches in the Control Panel.
  • If all mats disconnect, check the venue network or server status.
